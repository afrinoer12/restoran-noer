# RestoNoer - Website Restoran

Website restoran modern menggunakan HTML, CSS, dan JavaScript.

## Fitur

- Landing page restoran modern
- Navbar responsive
- Section menu makanan
- Filter kategori menu
- Keranjang pesanan
- Tambah/kurang jumlah pesanan
- Checkout otomatis ke WhatsApp
- Section tentang restoran
- Galeri restoran
- Kontak dan reservasi
- Responsive untuk desktop dan mobile

## Struktur Folder

```text
restoran-noer/
├── index.html
├── style.css
├── script.js
└── README.md
```

## Cara Menjalankan

1. Download atau clone project ini.
2. Buka folder `restoran-noer`.
3. Klik dua kali file `index.html`.
4. Web akan terbuka di browser.

## Custom

Untuk mengganti nomor WhatsApp, buka file `script.js`, lalu ubah:

```js
const phoneNumber = '6281234567890';
```

Untuk mengganti nama restoran, edit bagian `RestoNoer` di file `index.html`.
